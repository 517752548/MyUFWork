<div id="actionDesc">
<p>
Rotates a 2d Game Object on it's z axis so its forward vector points at a Target.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The GameObject to rotate.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Target Object</div>
	<div id="paramDesc">
The GameObject to Look At.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Rotation Offset</div>
	<div id="paramDesc">
Set the GameObject starting offset. In degrees. 0 if your object is facing right, 180 if facing left etc...</div>
</div>

<div id="paramRow">
	<div id="paramName">
Debug</div>
	<div id="paramDesc">
Draw a debug line from the GameObject to the Target.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Debug Line Color</div>
	<div id="paramDesc">
Color to use for the debug line.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Every Frame</div>
	<div id="paramDesc">
Repeat every frame.</div>
</div>
