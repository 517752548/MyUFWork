GUILayout Toolbar. NOTE: Arrays must be the same length as NumButtons or empty.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Num Buttons</div>
	<div id="paramDesc">
The number of buttons in the toolbar</div>
</div>

<div id="paramRow">
	<div id="paramName">
Selected Button</div>
	<div id="paramDesc">
Store the index of the selected button in an Integer Variable</div>
</div>

<div id="paramRow">
	<div id="paramName">
Button Events Array</div>
	<div id="paramDesc">
Event to send when each button is pressed.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Images Array</div>
	<div id="paramDesc">
Image to use on each button.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Texts Array</div>
	<div id="paramDesc">
Text to use on each button.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Tooltips Array</div>
	<div id="paramDesc">
Tooltip to use for each button.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Style</div>
	<div id="paramDesc">
A named GUIStyle to use for the toolbar buttons. Default is Button.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Every Frame</div>
	<div id="paramDesc">
Update the content of the buttons every frame. Useful if the buttons are using variables that change.</div>
</div>
