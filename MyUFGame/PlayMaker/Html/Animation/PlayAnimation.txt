<div id="actionDesc">
<p>
Plays an Animation on a Game Object. You can add named animation clips to the object in the Unity editor, or with the Add Animation Clip action.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
Game Object to play the animation on.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Anim Name</div>
	<div id="paramDesc">
The name of the animation to play.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Play Mode</div>
	<div id="paramDesc">
How to treat previously playing animations.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Blend Time</div>
	<div id="paramDesc">
Time taken to blend to this animation.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Finish Event</div>
	<div id="paramDesc">
Event to send when the animation is finished playing. NOTE: Not sent with Loop or PingPong wrap modes!</div>
</div>

<div id="paramRow">
	<div id="paramName">
Loop Event</div>
	<div id="paramDesc">
Event to send when the animation loops. If you want to send this event to another FSM use Set Event Target. NOTE: This event is only sent with Loop and PingPong wrap modes.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Stop On Exit</div>
	<div id="paramDesc">
Stop playing the animation when this state is exited.</div>
</div>
