<div id="actionDesc">
<p>
Unload Scene. Note that assets are currently not unloaded, in order to free up asset memory call Resources.UnloadUnusedAssets.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Scene Reference</div>
	<div id="paramDesc">
The reference options of the Scene</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene By Name</div>
	<div id="paramDesc">
The name of the scene to load. The given sceneName can either be the last part of the path, without .unity extension or the full path still without the .unity extension</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene At Build Index</div>
	<div id="paramDesc">
The build index of the scene to unload.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene At Index</div>
	<div id="paramDesc">
The index of the scene to unload.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene By Path</div>
	<div id="paramDesc">
The scene Path.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene By Game Object</div>
	<div id="paramDesc">
The GameObject unload scene of</div>
</div>

<div id="paramRow">
	<div id="paramName">
Unloaded</div>
	<div id="paramDesc">
True if scene was unloaded</div>
</div>

<div id="paramRow">
	<div id="paramName">
Unloaded Event</div>
	<div id="paramDesc">
Event sent if scene was unloaded </div>
</div>

<div id="paramRow">
	<div id="paramName">
Failure Event</div>
	<div id="paramDesc">
Event sent scene was not unloaded</div>
</div>
