<div id="actionDesc">
<p>
Loads the scene by its name or index in Build Settings. </p>
</div>
<div id="paramRow">
	<div id="paramName">
Scene Reference</div>
	<div id="paramDesc">
The reference options of the Scene</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene By Name</div>
	<div id="paramDesc">
The name of the scene to load. The given sceneName can either be the last part of the path, without .unity extension or the full path still without the .unity extension</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene At Index</div>
	<div id="paramDesc">
The index of the scene to load.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Load Scene Mode</div>
	<div id="paramDesc">
Allows you to specify whether or not to load the scene additively. See LoadSceneMode Unity doc for more information about the options.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Success</div>
	<div id="paramDesc">
True if the scene was loaded</div>
</div>

<div id="paramRow">
	<div id="paramName">
Success Event</div>
	<div id="paramDesc">
Event sent if the scene was loaded</div>
</div>

<div id="paramRow">
	<div id="paramName">
Failure Event</div>
	<div id="paramDesc">
Event sent if a problem occurred, check log for information</div>
</div>
