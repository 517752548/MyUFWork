<div id="actionDesc">
<p>
Move a GameObject from its current scene to a new scene. It is required that the GameObject is at the root of its current scene.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The Root GameObject to move to the referenced scene</div>
</div>

<div id="paramRow">
	<div id="paramName">
Find Root If Necessary</div>
	<div id="paramDesc">
Only root GameObject can be moved, set to true to get the root of the gameobject if necessary, else watch for failure events.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Success</div>
	<div id="paramDesc">
True if the merge succeeded</div>
</div>

<div id="paramRow">
	<div id="paramName">
Success Event</div>
	<div id="paramDesc">
Event sent if merge succeeded</div>
</div>

<div id="paramRow">
	<div id="paramName">
Failure Event</div>
	<div id="paramDesc">
Event sent if merge failed. Check log for information</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene Reference</div>
	<div id="paramDesc">
The reference option of the Scene</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene At Index</div>
	<div id="paramDesc">
The scene Index.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene By Name</div>
	<div id="paramDesc">
The scene Name.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene By Path</div>
	<div id="paramDesc">
The scene Path.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene By Game Object</div>
	<div id="paramDesc">
The Scene of GameObject</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene Found</div>
	<div id="paramDesc">
True if SceneReference resolves to a scene</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene Found Event</div>
	<div id="paramDesc">
Event sent if SceneReference resolves to a scene</div>
</div>

<div id="paramRow">
	<div id="paramName">
Scene Not Found Event</div>
	<div id="paramDesc">
Event sent if SceneReference do not resolve to a scene</div>
</div>
