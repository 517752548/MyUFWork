<div id="actionDesc">
<p>
Randomly flickers a Game Object on/off.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The GameObject to flicker.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Frequency</div>
	<div id="paramDesc">
The frequency of the flicker in seconds.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Amount On</div>
	<div id="paramDesc">
Amount of time flicker is On (0-1). E.g. Use 0.95 for an occasional flicker.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Renderer Only</div>
	<div id="paramDesc">
Only effect the renderer, leaving other components active.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Real Time</div>
	<div id="paramDesc">
Ignore time scale. Useful if flickering UI when the game is paused.</div>
</div>
