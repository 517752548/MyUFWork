<div id="actionDesc">
<p>
Gets a world direction Vector from 2 Input Axis. Typically used for a third person controller with Relative To set to the camera.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Horizontal Axis</div>
	<div id="paramDesc">
The name of the horizontal input axis. See Unity Input Manager.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Vertical Axis</div>
	<div id="paramDesc">
The name of the vertical input axis. See Unity Input Manager.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Multiplier</div>
	<div id="paramDesc">
Input axis are reported in the range -1 to 1, this multiplier lets you set a new range.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Map To Plane</div>
	<div id="paramDesc">
The world plane to map the 2d input onto.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Relative To</div>
	<div id="paramDesc">
Make the result relative to a GameObject, typically the main camera.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store Vector</div>
	<div id="paramDesc">
Store the direction vector.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store Magnitude</div>
	<div id="paramDesc">
Store the length of the direction vector.</div>
</div>
