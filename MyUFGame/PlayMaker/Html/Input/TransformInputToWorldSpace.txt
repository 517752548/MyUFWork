<div id="actionDesc">
<p>
Transforms 2d input into a 3d world space vector. E.g., can be used to transform input from a touch joystick to a movement vector.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Horizontal Input</div>
	<div id="paramDesc">
The horizontal input.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Vertical Input</div>
	<div id="paramDesc">
The vertical input.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Multiplier</div>
	<div id="paramDesc">
Input axis are reported in the range -1 to 1, this multiplier lets you set a new range.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Map To Plane</div>
	<div id="paramDesc">
The world plane to map the 2d input onto.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Relative To</div>
	<div id="paramDesc">
Make the result relative to a GameObject, typically the main camera.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store Vector</div>
	<div id="paramDesc">
Store the direction vector.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store Magnitude</div>
	<div id="paramDesc">
Store the length of the direction vector.</div>
</div>
