<div id="actionDesc">
<p>
Sends Events based on mouse interactions with a Game Object: MouseOver, MouseDown, MouseUp, MouseOff. Use Ray Distance to set how close the camera must be to pick the object.

NOTE: Picking uses the Main Camera.</p>
</div>
