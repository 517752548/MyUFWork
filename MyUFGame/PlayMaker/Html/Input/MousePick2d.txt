<div id="actionDesc">
<p>
Perform a Mouse Pick on a 2d scene and stores the results. Use Ray Distance to set how close the camera must be to pick the 2d object.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Store Did Pick Object</div>
	<div id="paramDesc">
Store if a GameObject was picked in a Bool variable. True if a GameObject was picked, otherwise false.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store Game Object</div>
	<div id="paramDesc">
Store the picked GameObject in a variable.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store Point</div>
	<div id="paramDesc">
Store the picked point in a variable.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Layer Mask</div>
	<div id="paramDesc">
Pick only from these layers.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Invert Mask</div>
	<div id="paramDesc">
Invert the mask, so you pick from all layers except those defined above.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Every Frame</div>
	<div id="paramDesc">
Repeat every frame.</div>
</div>
