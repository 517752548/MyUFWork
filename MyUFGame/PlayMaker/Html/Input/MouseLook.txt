<div id="actionDesc">
<p>
Rotates a GameObject based on mouse movement. Minimum and Maximum values can be used to constrain the rotation.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The GameObject to rotate.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Axes</div>
	<div id="paramDesc">
The axes to rotate around.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Sensitivity X</div>
	<div id="paramDesc">
Sensitivity of movement in X direction.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Sensitivity Y</div>
	<div id="paramDesc">
Sensitivity of movement in Y direction.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Minimum X</div>
	<div id="paramDesc">
Clamp rotation around X axis. Set to None for no clamping.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Maximum X</div>
	<div id="paramDesc">
Clamp rotation around X axis. Set to None for no clamping.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Minimum Y</div>
	<div id="paramDesc">
Clamp rotation around Y axis. Set to None for no clamping.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Maximum Y</div>
	<div id="paramDesc">
Clamp rotation around Y axis. Set to None for no clamping.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Every Frame</div>
	<div id="paramDesc">
Repeat every frame.</div>
</div>
