<div id="actionDesc">
<p>
Sets the position and rotation of the body. A GameObject can be set to control the position and rotation, or it can be manually expressed.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The target. An Animator component is required</div>
</div>

<div id="paramRow">
	<div id="paramName">
Target</div>
	<div id="paramDesc">
The gameObject target of the ik goal</div>
</div>

<div id="paramRow">
	<div id="paramName">
Position</div>
	<div id="paramDesc">
The position of the ik goal. If Goal GameObject set, position is used as an offset from Goal</div>
</div>

<div id="paramRow">
	<div id="paramName">
Rotation</div>
	<div id="paramDesc">
The rotation of the ik goal.If Goal GameObject set, rotation is used as an offset from Goal</div>
</div>

<div id="paramRow">
	<div id="paramName">
Every Frame</div>
	<div id="paramDesc">
Repeat every frame.</div>
</div>
