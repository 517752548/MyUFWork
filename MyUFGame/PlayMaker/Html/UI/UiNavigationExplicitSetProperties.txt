<div id="actionDesc">
<p>
Sets the explicit navigation properties of a UI Selectable component. Note that it will have no effect until Navigation mode is set to 'Explicit'.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The GameObject with the UI Selectable component.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Select On Down</div>
	<div id="paramDesc">
The down Selectable. Leave as None for no effect</div>
</div>

<div id="paramRow">
	<div id="paramName">
Select On Up</div>
	<div id="paramDesc">
The up Selectable.  Leave as None for no effect</div>
</div>

<div id="paramRow">
	<div id="paramName">
Select On Left</div>
	<div id="paramDesc">
The left Selectable.  Leave as None for no effect</div>
</div>

<div id="paramRow">
	<div id="paramName">
Select On Right</div>
	<div id="paramDesc">
The right Selectable.  Leave as None for no effect</div>
</div>

<div id="paramRow">
	<div id="paramName">
Reset On Exit</div>
	<div id="paramDesc">
Reset when exiting this state.</div>
</div>
