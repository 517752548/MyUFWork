<div id="actionDesc">
<p>
Sets various properties of a UI Layout Element component.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The GameObject with the UI LayoutElement component.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Min Width</div>
	<div id="paramDesc">
The minimum width this layout element should have.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Min Height</div>
	<div id="paramDesc">
The minimum height this layout element should have.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Preferred Width</div>
	<div id="paramDesc">
The preferred width this layout element should have before additional available width is allocated.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Preferred Height</div>
	<div id="paramDesc">
The preferred height this layout element should have before additional available height is allocated.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Flexible Width</div>
	<div id="paramDesc">
The relative amount of additional available width this layout element should fill out relative to its siblings.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Flexible Height</div>
	<div id="paramDesc">
The relative amount of additional available height this layout element should fill out relative to its siblings.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Every Frame</div>
	<div id="paramDesc">
Repeats every frame</div>
</div>
