<div id="actionDesc">
<p>
Tweens the color of the CanvasRenderer color associated with this Graphic.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The GameObject with a UI component.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Color</div>
	<div id="paramDesc">
The Color target of the UI component. Leave to none and set the individual color values, for example to affect just the alpha channel</div>
</div>

<div id="paramRow">
	<div id="paramName">
Red</div>
	<div id="paramDesc">
The red channel Color target of the UI component. Leave to none for no effect, else it overrides the color property</div>
</div>

<div id="paramRow">
	<div id="paramName">
Green</div>
	<div id="paramDesc">
The green channel Color target of the UI component. Leave to none for no effect, else it overrides the color property</div>
</div>

<div id="paramRow">
	<div id="paramName">
Blue</div>
	<div id="paramDesc">
The blue channel Color target of the UI component. Leave to none for no effect, else it overrides the color property</div>
</div>

<div id="paramRow">
	<div id="paramName">
Alpha</div>
	<div id="paramDesc">
The alpha channel Color target of the UI component. Leave to none for no effect, else it overrides the color property</div>
</div>

<div id="paramRow">
	<div id="paramName">
Duration</div>
	<div id="paramDesc">
The duration of the tween</div>
</div>

<div id="paramRow">
	<div id="paramName">
Ignore Time Scale</div>
	<div id="paramDesc">
Should ignore Time.scale?</div>
</div>

<div id="paramRow">
	<div id="paramName">
Use Alpha</div>
	<div id="paramDesc">
Should also Tween the alpha channel?</div>
</div>
